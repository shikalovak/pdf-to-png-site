# PDF to PNG — Chrome Extension

Convert PDF pages to PNG images directly in your browser — fast, simple, and private.
This repository contains the website source code. The extension itself is distributed via the Chrome Web Store.

**Website:** http://pdf-to-png.com?utm_source=readme  
**Chrome Web Store:** https://chromewebstore.google.com/detail/bgpfjcfimjfnlogfkpfckakdfbjabdfd?utm_source=readme

## How it works

1. Open the extension
2. Click **Choose PDF file**
3. Get PNG images (one per page)

---

## Privacy

Your PDF is processed locally in your browser. No uploads by default.

---

## FAQ

### Does it upload my PDF anywhere?
No — conversion happens locally on your device.

### Does it work offline?
Yes (once installed).

### How do I send feedback or feature requests?
Use the **Feedback / feature requests** link inside the extension.
